# 青春好肌動 O2O 快閃活動交付作業規劃

## 交付作業定義 (Deliverables)

本週結束後，各組需於下週上課前將作品上傳至課程平台。交付物將作為 Sprint 3 核心實戰評量。青春好肌動小組需完成兩份核心成果：一份能說清楚活動體驗與 O2O 閉環的一頁企劃書，以及一份能推導預算、轉換漏斗、ROI 與損益兩平點的財務預算表。

本次快閃活動不只是把產品帶到現場試吃，而是要證明「人寵共食」可以透過實體體驗建立信任，再把飼主導回線上會員、官網下單與推薦碼分享。活動必須同時具備可拍照、可互動、可掃碼、可回購四個條件。

## 1. O2O 快閃活動一頁企劃書

### 活動名稱

**青春好肌動｜毛孩共食快閃補給站**

### 活動核心口號

**和毛孩一起吃得健康，也一起被看見。**

### 活動概念

青春好肌動在高雄駁二特區設置兩日限定寵物友善快閃攤位，讓飼主帶毛孩完成拍照、試吃、互動任務與掃碼領券。現場用蔬菜比熊犬造型餐、手繪罐頭展示牆與人寵合照打卡區吸引停留，再透過 QR Code 將顧客導回 LINE、官網首購頁與社群推薦碼。

活動的核心不是一次性賣出最多產品，而是讓飼主留下數位足跡，建立後續回購、推薦與會員經營的基礎。

### 目標客群

1. **寵物飼主與毛孩陪伴族群**  
   重視安心成分、陪伴感、可愛體驗與人寵共同生活儀式。

2. **寵物友善生活消費族群**  
   常出入寵物友善咖啡廳、市集、展會、駁二文創場域，願意為生活風格與毛孩體驗付費。

3. **社群分享與新奇體驗族群**  
   容易被視覺造型、打卡牆、限量活動與推薦碼任務吸引，適合帶動二次擴散。

### 實體 5 感體驗設計

| 感官 | 現場設計 | O2O 轉換目的 |
| --- | --- | --- |
| 視覺 | 蔬菜比熊犬造型餐、手繪罐頭展示牆、人寵合照打卡區 | 吸引路過人流停留與拍照分享 |
| 嗅覺 | 清爽蔬菜香氣、低油低負擔的健康餐氣味 | 建立產品健康與安心印象 |
| 味覺 | 飼主試吃植物性健康餐，毛孩取得寵物友善試吃包 | 讓產品價值從概念變成體驗 |
| 觸覺 | 罐頭包裝、推薦碼吊牌、毛孩拍照小道具 | 增加互動感與記憶點 |
| 聽覺 | 現場「毛孩健康任務」引導話術與拍照倒數聲 | 引導顧客完成掃碼與分享任務 |

### O2O 閉環路徑圖

```text
社群預熱貼文 / 寵物 KOL 預告
        ↓
駁二現場看到快閃攤位
        ↓
人寵合照 + 試吃 + 毛孩健康任務
        ↓
掃 QR Code 加入 LINE 領首購優惠
        ↓
進入官網下單頁購買體驗組
        ↓
上傳人寵合照並使用推薦碼分享
        ↓
好友首購 / 本人回購 / LINE 會員再行銷
```

## 2. 財務轉換漏斗預算表

### 活動基本假設

| 項目 | 設定 |
| --- | --- |
| 活動地點 | 高雄駁二特區 |
| 活動天數 | 2 天 |
| 總預算 | NT$50,000 |
| 主要商品 | 青春好肌動人寵共食體驗組 |
| 平均客單價 (AOV) | NT$399 |
| 轉換目標 | 現場體驗後掃碼加入 LINE 並導向官網下單 |

### NT$50,000 預算分配

| 預算項目 | 金額 | 用途說明 |
| --- | ---: | --- |
| 快閃攤位與視覺佈置 | NT$15,000 | 攤位租借、手繪主視覺、展示桌、拍照牆、品牌旗幟 |
| 試吃包與產品樣品 | NT$9,000 | 飼主試吃、人寵共食體驗包、毛孩友善試吃包 |
| 現場人力與活動執行 | NT$8,000 | 兩日人員、排隊引導、拍照協助、試吃說明 |
| 社群廣告與 KOL 預熱 | NT$8,000 | Instagram 貼文、短影音、寵物 KOL 預告與活動提醒 |
| QR Code 與 LINE 優惠券 | NT$4,000 | Landing Page、LINE 加好友券、推薦碼歸因設定 |
| 拍照道具與打卡小物 | NT$4,000 | 毛孩吊牌、手拿牌、貼紙、打卡任務小禮 |
| 預備金 | NT$2,000 | 雨備、臨時耗材、現場設備補強 |
| **合計** | **NT$50,000** |  |

### O2O 轉換漏斗推導

| 漏斗階段 | 假設數值 | 計算方式 | 推導結果 |
| --- | ---: | --- | ---: |
| 駁二兩日路過人流 | 5,000 人 | 活動場域預估 | 5,000 人 |
| 進店率 | 18% | 5,000 × 18% | 900 人 |
| 體驗參與率 | 70% | 900 × 70% | 630 人 |
| LINE / QR 掃碼率 | 45% | 630 × 45% | 284 人 |
| 掃碼後購買率 | 18% | 284 × 18% | 51 筆訂單 |
| 平均客單價 | NT$399 | 每筆訂單平均金額 | NT$399 |
| 預估營收 | - | 51 × 399 | NT$20,349 |

### ROI 與損益兩平點

| 指標 | 公式 | 結果 |
| --- | --- | ---: |
| 預估營收 | 51 筆 × NT$399 | NT$20,349 |
| 活動總預算 | 固定支出 | NT$50,000 |
| 課程版 ROI | (20,349 - 50,000) / 50,000 | -59.3% |
| 損益兩平訂單數 | 50,000 / 399 | 126 筆 |
| 損益兩平購買率 | 126 / 284 | 44.4% |

### 財務結論

如果只看兩天快閃活動的現場首購，青春好肌動預估無法直接回收 NT$50,000 成本。這代表本次 O2O 快閃的真正價值不應只用單次現場營收判斷，而應把 LINE 名單、推薦碼分享、社群內容與後續回購納入評估。

因此，本組的策略重點應放在三件事：

1. **提高掃碼率**  
   每位完成試吃與拍照任務的飼主，都必須被引導加入 LINE 領取首購券。

2. **提高客單價**  
   現場優惠不主打單罐，而是主推體驗組、毛孩生日組與人寵共享組。

3. **提高回購與推薦**  
   讓現場顧客取得專屬推薦碼，後續好友首購與本人回購都能追蹤來源。

## 交付格式建議

### A. 一頁企劃書

建議做成橫式 A4 或 16:9 一頁圖像企劃。畫面要能在 60 秒內讓評審看懂：活動是什麼、誰會來、現場玩什麼、如何導回線上。

### B. 財務轉換漏斗預算表

建議使用 Google Sheets、Excel 或表格型 PDF。表格必須清楚呈現：

1. NT$50,000 預算分配
2. 人流到成交的每一層轉換率
3. 預估營收
4. ROI
5. 損益兩平訂單數
6. 損益兩平購買率
